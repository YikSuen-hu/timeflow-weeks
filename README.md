TimeFlow Weeks Manager

一个基于 React + Tailwind CSS 的时间管理工具，专为 Weeks 手账体系设计。

功能特性

双计时器：支持主任务与并行副任务同时计时。

Weeks 打印布局：自动生成符合 4mm 网格系统的周报表。

两种视图：极简实际时间条 + 计划/实绩对比图。

本地存储：数据保存在浏览器 LocalStorage 中，不会丢失。

本地运行

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

构建项目：

```bash
npm run build
```
